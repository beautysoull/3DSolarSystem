Инструкции для компьютеров:

  • Левая кнопка мыши: Двигать камеру вверх/вниз, влево/вправо.
  • Правая кнопка мыши: Двигать камеру вперёд/назад.
  • Колесо мыши: Увеличить/уменьшить масштаб.

Инструкции для мобильных устройств:

  • Один палец: Проведите вверх/вниз, влево/вправо для вращения камеры.
  • Два пальца: Пинч (сведение/разведение пальцев) для увеличения/уменьшения масштаба.
  • Свайп двумя пальцами: Перемещение камеры по экрану.
